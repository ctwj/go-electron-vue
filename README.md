# go-astilectron vue3 demo
Go Asticlectron with vuejs example app

go-astilectron v0.29.0
vue3
vite
typescript

# 安装依赖
```
cd frontend
yarn
```

# 编译
```
./build.sh
```

# 开启调试 
`ctrl+d`

# 关闭调试
main.go， 修改 debug = false

# 修改窗口属性
```
# WindowOptions
https://pkg.go.dev/github.com/asticode/go-astilectron#WindowOptions
```
